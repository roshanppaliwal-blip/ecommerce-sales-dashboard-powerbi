# 📊 Flipkart Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes Flipkart sales data to identify key business insights such as revenue trends, customer behavior, and regional performance.

## 🛠️ Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis) [if used]
- Power BI (Dashboard & Visualization)

## 📊 Key KPIs
- Total Revenue
- Total Orders
- Average Order Value (AOV)
- Top Categories & Products
- Regional Performance

## 🔍 Key Insights
- Laptops generate highest revenue due to higher price per unit
- North region shows strongest sales performance
- Certain categories have higher return rates

## 📈 Features
- Interactive filters (Region, Category)
- Drill-down analysis
- Dynamic KPI cards
