# 📊 Sales Analysis Project

## 🔍 Objective
Analyze sales data to identify trends, top products, and regional performance.

## 📁 Dataset
- Source: Sample dataset
- Columns: Date, Product, Region, Sales

## 🧹 Data Cleaning
- Removed missing values
- Standardized product & region names

## 📊 Analysis Performed
- Revenue calculation
- Product-wise performance
- Region-wise comparison

## 📈 Key Insights
- Laptop generates highest revenue
- North region performs best
- Opportunity to improve South region sales

## 🛠 Tools Used
- Excel (Pivot Tables, Formulas)
